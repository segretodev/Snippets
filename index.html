Códigos HTML

Título e Parágrafo

Esse código cria um título de nível 2 e um parágrafo em uma página HTML:

<h2>Título</h2> 
    <p>Parágrafo</p>

Lista Ordenada

Esse código cria uma lista ordenada com três itens em uma página HTML:


<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Imagem

Esse código insere uma imagem na página HTML:

<img src="caminho_da_imagem" alt="texto_alternativo">
